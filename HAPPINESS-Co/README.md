# HAPPINESS&Co 
HAPPINESS & Co es una empresa joven y dinámica dedicada al mundo del ocio.
Ante la inmensa marea de actividades culturales, esta plataforma surge como una agenda diseñada para recopilar y organizar todos los eventos de la localidad e inmediaciones.

## Características del Proyecto
* Interfaz Intuitiva: Diseño limpio basado en la identidad visual de la marca.
* Categorización por Colores: Navegación facilitada por códigos de color corporativos para cada tipo de evento.
* Secciones Especializadas: Páginas dedicadas para Música, Escena (Teatro/Cine), Social (Exposiciones) y Extras.
* Sistema de Inscripción: Funcionalidad de registro para eventos mediante ventanas modales.
* Diseño Adaptable: Optimizado para una visualización correcta en distintos dispositivos.

## Identidad Visual 
El proyecto utiliza los siguentes colores.
* Rosa (#ba007c): Color corporativo principal y para eventos musicales.
* Azul (#009bdb): Color para cine, teatro y artes escénicas.
* Naranja (#ff7b00): Color para exposiciones y eventos sociales.
* Gris (#555555): Color para otros eventos.

## Estructura del Sitio
* index.html: Página principal con acceso a las rutas culturales.
* eventos.html: Listado general con sistema de inscripción.
* empresa.html: Información sobre la misión y valores de la empresa.
* contacto.html: Formulario de contacto y atención al usuario.
* Categorías: musica.html, escena.html, social.html y extras.html.

## Tecnologías Utilizadas
* HTML5: Estructura semántica del contenido.
* CSS3: Estilos personalizados, Flexbox y variables de color.
* JavaScript: Gestión de navegación, modales y validación de formularios.

## Instalación y Uso
1. Descargar o clonar el repositorio.
2. Mantener la estructura de carpetas de activos (Assets/img).
3. Ejecutar el archivo index.html en un navegador web.
