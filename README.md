# HAPPINESS&Co 
Este es un proyecto para organizar los eventos culturales de la ciudad (musica, teatro y exposiciones) en una web facil de usar.

## Colores del Proyecto
Usamos colores para que se vea rapido que tipo de evento es:
  Rosa (#ba007c): Musica y marca principal
  Azul (#009bdb): Cine y Teatro
  Naranja (#ff7b00): Exposiciones
  Gris (#555555): Otros eventos

## Que hace la web
  Muestra eventos proximos por categorias
  Tiene un historial para ver fotos de eventos que ya pasaron
  Permite guardar eventos en una lista de favoritos
  Funciona en moviles y ordenadores

## Herramientas usadas
  HTML:  Para el diseño y la estructura de las paginas
  MySQL: Para guardar los usuarios y los eventos 
  Figma: Para el diseño previo de las pantallas

## Archivos importantes
  index.html: La pagina principal
  eventos.html: Lista de todos los eventos
  Base datos Happiness&Co.sql: El archivo para montar la base de datos 
  Assets: Carpeta con las fotos y el logo.
